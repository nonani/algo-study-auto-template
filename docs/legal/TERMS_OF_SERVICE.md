# 이용약관 (Terms of Service)

최종 업데이트: 2026-02-21

## 1. 목적
본 서비스는 알고리즘 스터디 운영을 위해 solved.ac에서 문제 정보를 조회하고, 사용자가 권한을 부여한 Notion 데이터베이스에 주간 문제를 자동 등록합니다.

## 2. 제공 기능
- 주간(일요일 기준) 문제 자동 등록
- 중복 문제 방지 로직
- Notion 데이터베이스 속성 매핑 기반 입력

## 3. 사용자 책임
- Notion 통합 권한을 적절한 데이터베이스에만 부여해야 합니다.
- GitHub Secrets 등 민감정보를 안전하게 보관해야 합니다.
- 자동 등록 결과를 주기적으로 검토해야 합니다.

## 4. 서비스 제한
- Notion API, solved.ac API, GitHub Actions의 장애/변경에 따라 기능이 일시 중단될 수 있습니다.
- 외부 API 정책 변경 시 동작이 달라질 수 있습니다.

## 5. 보증의 부인
본 서비스는 "있는 그대로" 제공되며, 특정 목적 적합성 및 무중단 운영을 보증하지 않습니다.

## 6. 책임의 제한
서비스 사용으로 발생한 직접/간접 손해에 대해 법이 허용하는 범위 내에서 책임이 제한됩니다.

## 7. 변경
약관은 필요 시 변경될 수 있으며, 변경 시 본 문서의 업데이트 일자를 갱신합니다.
