# 개인정보처리방침 (Privacy Policy)

최종 업데이트: 2026-02-21

## 1. 수집하는 정보
본 자동화 도구는 서비스 제공을 위해 다음 정보를 처리할 수 있습니다.
- Notion API 토큰(사용자 제공, GitHub Secrets 등 비공개 저장소에 보관)
- Notion 페이지/데이터베이스 식별자
- 문제 등록 결과 로그(문제 번호, 제목, 실행 시각 등)

본 도구는 회원가입 정보를 자체적으로 수집/저장하지 않습니다.

## 2. 정보 이용 목적
- Notion 데이터베이스에 주간 문제 등록
- 중복 등록 방지 및 실행 상태 확인

## 3. 보관 및 파기
- 토큰과 설정값은 사용자가 구성한 비밀 저장소(GitHub Secrets 등)에 보관됩니다.
- 로그 보관 기간은 사용자의 GitHub Actions 로그 정책을 따릅니다.

## 4. 제3자 제공
서비스 동작을 위해 아래 외부 서비스와 통신합니다.
- Notion API
- solved.ac API
- GitHub Actions

법령에 따른 경우를 제외하고 별도의 제3자 제공을 하지 않습니다.

## 5. 안전성 조치
- API 토큰은 코드에 하드코딩하지 않고 환경변수/시크릿으로 관리합니다.
- 최소 권한 원칙에 따라 필요한 데이터베이스에만 통합 권한을 부여해야 합니다.

## 6. 이용자 권리
이용자는 언제든지 통합 권한을 해제하고 토큰을 폐기할 수 있습니다.
